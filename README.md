# OSPF-LEARNING-1

This project demonstrates the configuration of Open Shortest Path First (OSPF) in a small network using four routers.
The lab shows how routers form OSPF neighbors and exchange routes dynamically.

The configuration uses interface-based OSPF configuration and assigns a router ID to each router.
Here the cost are same so the packets are can be forwarded from both the paths.
DR/BDR are selected on basiss of id i have not provided any priority.
